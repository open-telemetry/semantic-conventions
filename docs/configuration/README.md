# Configuration

This section describes semantic convention related configuration options available in
[OpenTelemetry Declarative Configuration](https://opentelemetry.io/docs/specs/otel/configuration/).

Declarative configuration provides a file-based, structured way to configure
OpenTelemetry SDKs and instrumentations. For more details, see the
[configuration specification](https://opentelemetry.io/docs/specs/otel/configuration/)
and the [configuration file schema](https://github.com/open-telemetry/opentelemetry-configuration).

* [Version selection](version-selection.md): How to select semantic convention versions.
