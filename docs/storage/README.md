**Status**: [Development][DocumentStatus]

This document defines semantic conventions for storage systems and their components.

Storage semantic conventions cover various types of storage systems including:
- Block storage system
- File storage systems
- Object storage systems
- Software-defined storage platforms
- Tape libraries and backup systems
